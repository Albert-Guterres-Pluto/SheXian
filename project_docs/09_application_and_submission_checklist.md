# 立项与提交材料对照清单

本文件根据 `need` 文件夹中的学校附件整理，用于填写立项申请、准备中期反馈和结项材料。不要修改 `need` 文件夹中的原始需求文档。

## 一、立项申请书填写要点

附件2《北京邮电大学社会实践项目立项申请书》要求填写：

1. 基本情况：主题、团队名称、负责人信息、指导教师、团队成员和工作分配。
2. 实践内容策划：1000字以内，包括活动时间、地点、流程、时间表和预算。
3. 预期目标及成果：500字以内。
4. 安全预案：500字以内。
5. 可行性分析：500字以内。

## 二、立项申请书可粘贴文本

### 实践内容策划（1000字以内）

本项目拟由北京邮电大学5人实践团队赴安徽省黄山市歙县富堨镇承狮村开展暑期社会实践，围绕承狮村麒麟舞传统，开展非遗文化挖掘、短视频传播和特色实践路线设计。项目计划实地实践5至6天，主要地点包括歙县北站、富堨镇、承狮村及歙县县城游客集中区域。

实践流程分为三个阶段。第一阶段为出发前准备：联系歙县文旅部门、富堨镇和承狮村，核实麒麟舞传承人、表演队、排练或展演时间；完成访谈提纲、简版问卷、拍摄脚本、授权书、安全责任书、家长/监护人知情同意书、保险和应急预案。第二阶段为实地调研：D1抵达歙县北站和承狮村，拜访镇村干部并踏勘路线；D2深访麒麟舞传承人，记录历史源流、传承谱系、动作、道具和锣鼓；D3开展村落空间观察和村民访谈，记录麒麟舞与祠堂、广场、巷道等公共空间的关系；D4访问文化站、学校或商户，发放简版问卷，梳理传承和传播难点；D5在歙县县城或游客点位调研游客对“从歙县北站到承狮村看非遗”的接受度，并补拍路线素材；D6视情况开展补拍、初剪反馈和离村前成果确认。第三阶段为返校后整理：完成2至3分钟横屏综合短片、3至5支竖屏短视频、调研报告、论文或论文式实践总结、访谈记录、照片视频、简版路线建议和短视频发布建议。

经费预算按5人、5至6天估算，建议预算约20000元，主要包括往返交通4000元、当地交通1800元、住宿3600元、餐费2400元、保险药品500元、调研材料600元、传承人及展演支持2500元、拍摄耗材1500元、后期制作1000元、简版物料600元、机动费用1500元。若地方可协助住宿、交通或排练安排，压缩版预算约14250元。

### 预期目标及成果（500字以内）

项目目标包括：第一，完成承狮村麒麟舞基础田野调查，梳理其历史源流、传承现状、表演场景、道具服饰、锣鼓音乐和村落关系；第二，探索短视频时代下麒麟舞如何走出富堨、走出歙县，形成适合地方使用的轻量传播建议；第三，拍摄1支2至3分钟横屏综合短片和3至5支竖屏短视频，满足学校结项和地方传播需要；第四，设计“歙县北站至承狮村非遗体验”简版路线建议，为当地文旅和研学活动提供参考。

预期成果包括：论文或论文式实践总结1篇、调研报告1份、访谈记录10至15份、问卷数据约80至120份、2至3分钟横屏综合短片1支、竖屏短视频3至5支、结项电子照片不少于10张、承狮村麒麟舞基础资料目录1套、简版路线建议1套、短视频发布建议1份。

### 安全预案（500字以内）

团队出发前完成学校审批、安全培训、保险购买、成员安全责任书和家长/监护人知情同意书收集。实践期间实行队长负责制和每日早晚点名制度，外出至少两人同行，不夜间单独行动。暑期高温多雨，户外调研和拍摄尽量安排在早晚，携带防暑、防蚊、常用药和充电宝；遇暴雨、雷电、地质灾害预警等情况，立即停止户外活动，转为室内访谈或资料整理。交通方面优先使用正规公共交通或合规车辆，不乘坐无资质车辆。拍摄和访谈前取得受访者同意，涉及未成年人、宗祠、文物空间和不宜公开内容时严格执行授权和避让。团队不擅自变更实践地点和计划，不组织商业性活动，不给基层增加额外负担。若发生安全、舆情或突发事件，先保障人身安全，及时联系指导老师、学院和地方联系人。

### 可行性分析（500字以内）

项目可行性主要体现在四方面。第一，选题明确。需求表中富堨镇课题要求围绕麒麟舞开展非遗文化挖掘、短视频传播和宣传短视频拍摄，本项目聚焦承狮村麒麟舞，方向具体、边界清晰。第二，团队规模适中。5人小队采用“一人主责、多岗协同”分工，分别负责地方联络、调研文字、影像后期、传播数据和安全财务，成果设置为轻量化，避免超出执行能力。第三，方法匹配。项目综合使用口述史访谈、参与式观察、简版问卷、影像记录、短视频内容分析和路线试走，能够支撑论文、报告、照片视频等成果。第四，地方价值明确。项目把承狮村麒麟舞与歙县北站区位结合，提出“下高铁看非遗”的传播入口，并向地方交付基础资料目录、视频素材、路线建议和发布建议，有助于后续非遗保护和乡村文旅传播。

## 四、团队介绍（400字以内）

北京邮电大学“邮见承狮”实践团拟赴安徽省黄山市歙县富堨镇承狮村，围绕当地麒麟舞传统开展暑期社会实践。团队暂定5名学生，结合北邮信息通信、数字媒体、网络传播和数据分析特色，采用口述史访谈、参与式观察、简版问卷、影像记录和短视频内容分析等方法，挖掘麒麟舞历史文化内涵、传承现状和传播转化路径。实践期间，团队将重点走访传承人、村干部、村民、游客和文旅相关主体，拍摄2至3分钟横屏综合短片和3至5支竖屏短视频，设计“歙县北站至承狮村非遗体验”简版路线建议。项目坚持成果少而实，重视真实性、授权合规和地方实际需求，力求为承狮村麒麟舞保护传承和歙县乡村文旅传播提供可参考的青年方案。

## 五、中期反馈材料

“集邮广益”机制专项项目必须提交中期成果反馈材料，建议按以下清单准备：

1. 300字左右总结新闻，涵盖背景、主题、时间、地点、活动内容、服务群体和阶段性成果，尽量有具体数据。
2. 8至12张过程性照片，体现访谈、调研、拍摄、路线试走、村落观察等真实实践过程。
3. 照片单张不超过10M，以“序号-实践内容介绍”命名。
4. 减少座谈、摆拍和纯合影照片，避免墨镜、防晒帽遮挡主要人物。
5. 提交前由带队教师或学院团委审核。
6. 文件夹命名建议：“学院全称-团队名全称-队长姓名-队长联系方式”。

## 六、结项材料

结项电子版材料建议按“材料名称-团队名全称”命名，并集中打包：

1. 《北京邮电大学社会实践结项登记表》。
2. 2至5分钟高质量MP4视频，建议横屏16:9，分辨率不低于1920x1080。
3. 至少10张电子版照片，建议JPG格式，以活动内容命名。
4. 结项登记表中精选3至5张照片，至少5张电子照片有团队成员出镜。
5. 每名队员200字左右实践感悟。
6. 800字左右社会实践报告，说明项目简介、实践过程、社会价值和成果。
7. 调研报告、论文或论文式实践总结、访谈记录、问卷数据、视频链接或素材索引。
8. 感谢信、表扬信、纸质媒体报道、实践地证明等材料如有则单独整理。

## 七、评优加分但不硬性承诺的事项

1. 校内或地方官方媒体报道。
2. 围绕实践地发展需求撰写资政建议并递交属地机关。
3. 实践地出具感谢信或表扬信。
4. 与承狮村、富堨镇或相关单位协商长期实践基地。
5. 后续申报创新创业或社会实践类竞赛。

以上事项可争取，但不写成必达目标，不以媒体曝光量作为学生参与或成果认定依据。
