# 北京邮电大学2026年“集邮广益”暑期专项社会实践立项准备资料

项目暂定方向：围绕安徽省黄山市歙县富堨镇承狮村麒麟舞传统，开展非遗文化挖掘、短视频传播、数字档案建设和乡村文旅路线共创。

## 当前核心判断

1. 项目应从“富堨镇非遗调研”收束为“承狮村麒麟舞个案研究与传播转化”。
2. 项目暂按5人小队设计，成果应轻量可完成，避免承诺大型展陈、长期运营或过多视频数量。
3. 承狮村与歙县北站的区位关系适合转化为传播钩子：“下了高铁，看见徽州麒麟舞”。
4. 实践成果不应只停留在调研报告，应向当地交付可继续使用的短视频素材、路线建议和基础资料目录。
5. 对“麒麟舞是否已列入某级非遗名录、具体传承谱系、历史起源”等信息，出发前和到达后必须向歙县文旅部门、富堨镇文化站、承狮村村委会及传承人核实。
6. `need` 文件夹中的附件为学校需求和表格模板，仅作为只读依据，不在本项目文档优化中修改。

## 对照需求附件后的硬性要求

| 阶段 | 要求 | 项目响应 |
|---|---|---|
| 立项 | 立项申请书需填写实践内容策划、预期目标及成果、安全预案、可行性分析 | 已在 `09_application_and_submission_checklist.md` 中整理可粘贴文本 |
| 中期反馈 | “集邮广益”项目必须提交300字左右总结新闻和8至12张过程性照片 | 已加入任务清单和拍摄要求 |
| 结项 | 需提交2至5分钟MP4视频，建议横屏16:9、分辨率不低于1920x1080 | 综合短片按2至3分钟横屏制作，竖屏短视频作为补充传播成果 |
| 结项 | 至少提交10张电子照片，结项登记表中精选3至5张，至少5张照片有实践队员出镜 | 已调整照片采集和素材清单 |
| 合规 | 不摆拍走秀、不夸大成果、不以媒体曝光量作为硬性认定依据、不擅自变更行程 | 已纳入风险预案和传播指标说明 |
