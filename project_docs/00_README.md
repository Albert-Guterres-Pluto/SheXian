项目暂定方向：围绕安徽省黄山市歙县富堨镇承狮村麒麟舞传统，开展非遗文化挖掘、短视频传播、数字档案建设和乡村文旅路线共创。

## 当前核心判断

1. 项目应从“富堨镇非遗调研”收束为“承狮村麒麟舞个案研究与传播转化”。
2. 承狮村与歙县北站的区位关系适合转化为传播钩子：“下了高铁，看见徽州麒麟舞”。
3. 实践成果不应只停留在调研报告，应向当地交付可继续使用的短视频素材、路线方案、账号运营建议和数字档案目录。
4. 对“麒麟舞是否已列入某级非遗名录、具体传承谱系、历史起源”等信息，出发前和到达后必须向歙县文旅部门、富堨镇文化站、承狮村村委会及传承人核实。
