# 风险预案

## 一、安全风险

| 风险 | 表现 | 预案 |
|---|---|---|
| 高温中暑 | 暑期户外拍摄、路线试走时间长 | 避开11:00至15:00户外高强度活动，携带藿香正气水、口服补液盐、防晒用品 |
| 暴雨天气 | 山区和乡村道路出行受影响 | 每天查看天气，雨天转为室内访谈、道具拍摄、资料整理 |
| 交通风险 | 村镇道路不熟，夜间出行风险高 | 统一出行，不夜间单独行动，优先使用正规车辆 |
| 设备搬运 | 摄影器材多，丢失或损坏 | 建立设备借用表，每天清点，重要设备不离人 |
| 食品和肠胃问题 | 队员饮食不适应 | 选择卫生条件稳定餐饮，准备常用药，避免生冷食物 |

## 二、文化表达风险

| 风险 | 表现 | 预案 |
|---|---|---|
| 历史源流误写 | 把传说写成事实，把待核实信息写成定论 | 使用“据受访者回忆”“待地方文献核实”等表述 |
| 非遗娱乐化 | 为流量过度剪辑、恶搞动作或音乐 | 发布前由传承人、村委会或文化站复核 |
| 禁忌内容公开 | 拍到不适合公开的仪式或空间 | 访谈前问清哪些可拍、哪些不可拍 |
| 过度承诺 | 承诺带来大量游客或经济收益 | 表述为“提供传播和路线建议”，不承诺不可控结果 |

## 三、授权风险

| 场景 | 要求 |
|---|---|
| 传承人正面访谈 | 录音、录像、肖像、姓名公开均需授权 |
| 未成年人出镜 | 必须取得监护人同意 |
| 宗祠和文物空间 | 先询问村委会和管理方，必要时只拍外景 |
| 表演队整体拍摄 | 由负责人确认团队授权，同时尊重个人拒绝出镜 |
| 网络发布 | 发布前再次确认可公开视频范围 |

## 四、调研风险

| 风险 | 表现 | 预案 |
|---|---|---|
| 访谈对象临时变动 | 传承人外出、队员无法集合 | 准备备选访谈对象和室内资料整理任务 |
| 问卷样本不足 | 游客量不足或拒答率高 | 线上补充青年问卷，县城游客点位补发 |
| 信息相互矛盾 | 不同受访者对起源说法不一 | 如实记录差异，不强行统一 |
| 材料零散 | 录音、照片、文字无法对应 | 每天整理素材索引和访谈编号 |

## 五、传播风险

| 风险 | 表现 | 预案 |
|---|---|---|
| 视频数据不理想 | 播放低、互动少 | 调整首屏、标题、封面和时长，优先推强视觉内容 |
| 评论争议 | 网友质疑真实性或商业化 | 统一回应口径，说明以传承人口述和地方核实为准 |
| 平台误判 | 音乐、画面或标题触发审核 | 使用自采声音或可授权音乐，避免夸张标题 |
| 地方账号不转发 | 缺少联动 | 提前准备适合官方发布的稳妥版本 |

## 六、成果质量风险

| 风险 | 表现 | 预案 |
|---|---|---|
| 主片素材不完整 | 缺少同期声、空镜或动作细节 | 拍摄前按清单逐项核对 |
| 报告空泛 | 只有描述，没有数据和案例 | 每天补充访谈原话、问卷数据、路线时间 |
| 地方用不上 | 成果偏学术，缺少可操作交付 | 交付短视频、标题库、路线表、二维码文案 |
| 后期拖延 | 返校后课程或事务冲突 | 出发前确定剪辑负责人和时间节点 |

## 七、应急联络机制

1. 每日早晚两次点名。
2. 外出至少两人同行。
3. 队员手机保持电量充足，携带充电宝。
4. 建立学校指导老师、队长、地方联系人、住宿联系人、车辆联系人通讯录。
5. 出现安全事件先保障人身安全，再通知指导老师和地方联系人。
