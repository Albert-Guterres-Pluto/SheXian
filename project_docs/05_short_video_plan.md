# 短视频策划方案

## 一、传播定位

核心定位：下了歙县北站，走进承狮村，看一场徽州麒麟舞。

目标受众：

1. 计划前往歙县、黄山、徽州地区的游客。
2. 对非遗、古村落、民俗表演感兴趣的青年用户。
3. 本地居民、返乡青年和地方文旅账号受众。
4. 北京邮电大学师生和高校社会实践关注者。

传播目标：

1. 让用户知道“承狮村有麒麟舞传统”。
2. 让用户理解“麒麟舞不只是表演，也是村落记忆和乡土文化”。
3. 让用户形成“到歙县可以顺路到承狮村体验”的初步兴趣。
4. 为承狮村和富堨镇留下少量可复用的短视频素材和发布建议。

说明：播放量、转发量、媒体报道量只作为内部复盘参考，不作为团队成员参与实践或结项认定的硬性指标。

## 二、平台策略

| 平台 | 内容重点 | 建议形式 |
|---|---|---|
| 抖音 | 强视觉、强节奏、强反差 | 15至45秒竖屏爆点视频 |
| 视频号 | 地方传播、人情连接、官方转发 | 30至90秒纪实短片 |
| 小红书 | 路线攻略、拍照点、体验感 | 图文笔记+短视频 |
| B站 | 文化解释、青年实践、纪录片 | 3至8分钟横版或竖版混剪 |
| 北邮官方平台 | 实践成果、青年担当、学科特色 | 图文报道+综合短片 |

## 三、核心话题

建议话题标签：

1. #承狮村麒麟舞
2. #歙县非遗
3. #下高铁看非遗
4. #徽州有麟
5. #北邮社会实践
6. #集邮广益

## 四、综合短片脚本

片名：下高铁，看见一只徽州麒麟

时长：2至3分钟。按结项要求制作横屏16:9 MP4版本，分辨率不低于1920x1080；竖屏短视频另按9:16制作。

| 段落 | 时长 | 内容 | 画面 |
|---|---:|---|---|
| 开场钩子 | 0-8秒 | “歙县北站旁，藏着一支麒麟舞。” | 高铁站、村口、麒麟头猛然入画、锣鼓起 |
| 地点建立 | 8-25秒 | 介绍承狮村、富堨镇、歙县徽州文化背景 | 村路、老屋、田野、路牌、村民生活 |
| 人物进入 | 25-75秒 | 传承人讲述自己如何学会麒麟舞 | 人物近景、手部、旧道具、排练 |
| 文化解释 | 75-120秒 | 解释麒麟寓意、动作、锣鼓、节庆场景 | 动作拆解、道具细节、锣鼓节奏 |
| 北邮行动 | 120-160秒 | 访谈、拍摄、路线试走、短视频传播 | 队员现场工作、问卷、地图 |
| 结尾转化 | 160-180秒 | “把承狮村的麒麟，寄给更多人。” | 麒麟舞高潮、村落空镜、路线提示 |

## 五、竖屏短视频选题

本项目暂按5人小队执行，优先完成前5条；若实地素材充足，再补充第6至8条。

| 编号 | 标题 | 时长 | 首屏钩子 | 核心内容 |
|---|---|---:|---|---|
| 1 | 歙县北站旁，藏着一支麒麟舞 | 20秒 | 麒麟头从巷口冲出 | 地点+表演瞬间 |
| 2 | 下了高铁，先去看一场非遗 | 30秒 | 高铁站切到锣鼓声 | 路线攻略 |
| 3 | 一只麒麟头，要几个人舞起来？ | 30秒 | 队员钻入麒麟道具 | 道具和角色 |
| 4 | 承狮村为什么会跳麒麟舞？ | 60秒 | 传承人一句话 | 起源与村落记忆 |
| 5 | 跟传承人学一个麒麟步 | 20秒 | 动作对比 | 互动挑战 |
| 6 | 锣鼓一响，村里人都出来了 | 30秒 | 锣鼓特写 | 现场氛围 |
| 7 | 徽州不只有古城，还有麒麟舞 | 45秒 | 古城切承狮村 | 县域路线联动 |
| 8 | 年轻人还愿意学麒麟舞吗？ | 60秒 | 青年队员回答 | 传承困境 |

## 六、拍摄清单

必拍画面：

1. 歙县北站外景、站牌、到村路线。
2. 承狮村村口、主路、广场、祠堂或公共活动空间。
3. 麒麟头、服饰、锣鼓、道具细节。
4. 传承人正面访谈、侧面工作、手部动作。
5. 排练或展演全景、中景、特写。
6. 村民观看、儿童学习、青年体验。
7. 队员调研、拍摄、问卷、路线试走。
8. 夕阳、晨雾、炊烟、田野等村落环境空镜。

结项照片要求：

1. 中期反馈准备8至12张过程性照片，以真实调研、拍摄、访谈、路线试走为主。
2. 结项至少提交10张电子照片，建议JPG格式。
3. 结项登记表中精选3至5张照片，至少5张电子照片应有实践团队成员出镜。
4. 照片命名建议采用“序号-实践内容介绍”，单张不超过10M。

慎拍或需授权画面：

1. 宗祠内部和文物细节。
2. 未成年人正脸。
3. 家庭室内空间。
4. 未经同意的村民近景。
5. 涉及祭祀、禁忌或传承人明确不希望公开的内容。

## 七、发布节奏

| 阶段 | 时间 | 内容 |
|---|---|---|
| 预热 | 出发前3至5天 | 项目介绍、为什么去承狮村 |
| 实地 | D1至D7 | 每天1条现场短视频或图文 |
| 第一轮传播 | 返校后7天内 | 3条爆点视频+1条路线视频 |
| 第二轮传播 | 返校后14天内 | 人物故事、道具科普、青年体验 |
| 成果发布 | 返校后30天内 | 综合短片、调研成果、路线建议 |

## 八、数据指标

| 指标 | 目标 |
|---|---:|
| 短视频发布数量 | 3至5条 |
| 综合短片 | 1支 |
| 横屏结项视频 | 2至5分钟MP4，1920x1080以上 |
| 中期过程照片 | 8至12张 |
| 结项电子照片 | 不少于10张 |
| 内部传播复盘 | 记录播放、完播、互动和评论反馈，不设硬性曝光目标 |
| 路线反馈 | 收集不少于20条游客或青年受众反馈 |

## 九、标题和封面原则

1. 首屏必须出现麒麟头、锣鼓、村口、传承人或高铁站中的至少一个强识别元素。
2. 标题尽量包含地点词：“歙县”“承狮村”“歙县北站”“徽州”。
3. 不使用“濒临失传”“最后一代”等未经证实的夸张表达。
4. 封面文字控制在8至14字。
5. 每条视频只讲一个重点，不把历史、路线、人物、成果全塞进一条视频。
6. 不使用虚构、夸大或猎奇表达，不把“媒体曝光”包装为实践成果本身。
