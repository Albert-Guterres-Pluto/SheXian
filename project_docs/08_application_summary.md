# 立项申报摘要

## 项目名称

邮见承狮：短视频时代歙县富堨镇承狮村麒麟舞数字传承实践

## 项目简介

本项目拟由北京邮电大学实践团队赴安徽省黄山市歙县富堨镇承狮村开展暑期社会实践。团队将以承狮村麒麟舞传统为核心对象，通过田野调查、口述史访谈、影像记录、短视频传播分析和文旅路线共创，系统挖掘麒麟舞的历史文化内涵、传承现状和传播转化路径。

项目结合北京邮电大学信息通信、数字媒体和网络传播特色，计划建立承狮村麒麟舞数字档案，拍摄宣传片和系列竖屏短视频，设计“歙县北站至承狮村非遗体验”特色路线，形成论文、调研报告、访谈记录、照片视频、短视频传播手册和地方成果包。项目目标是让承狮村麒麟舞被记录、被理解、被传播，并为富堨镇乡村文化振兴和歙县文旅资源拓展提供可执行方案。

## 选题意义

承狮村麒麟舞是富堨镇具有地方特色的民俗文化资源，承载着村落记忆、节庆习俗和乡土情感。当前，许多乡村非遗面临传承人老龄化、青年参与不足、线上表达薄弱、游客转化困难等问题。短视频平台为非遗传播提供了新入口，但也要求内容具备强视觉、强故事、强互动和可到达的路线设计。

本项目将承狮村麒麟舞置于歙县徽文化和乡村振兴背景下研究，既关注文化保护，也关注传播转化和地方实际需求，具有较强的实践价值。

## 主要内容

1. 走访歙县文旅部门、富堨镇、承狮村，核实麒麟舞历史源流、名录身份、传承人和表演场景。
2. 深度采访传承人、表演队员、锣鼓师、村干部、村民、游客和文旅从业者。
3. 记录麒麟舞动作、音乐、道具、服饰、排练和展演过程。
4. 分析短视频平台上非遗传播规律，设计适合承狮村麒麟舞的内容矩阵。
5. 拍摄1支3至5分钟宣传片和8至12支竖屏短视频。
6. 设计半日线、一日线和两日研学线，推动麒麟舞与歙县县域文旅资源联动。
7. 向当地交付数字档案目录、短视频素材包、路线方案、运营日历和调研报告。

## 实践方法

项目采用口述史访谈、参与式观察、问卷调查、视觉民族志、短视频内容分析、路线试走和共创工作坊等方法。团队将在实地期间完成资料采集和初步影像制作，返校后完成数据分析、报告撰写、视频后期和成果反馈。

## 预期成果

1. 论文1篇。
2. 调研报告1份。
3. 访谈记录20至30份。
4. 问卷数据约200份。
5. 主宣传片1支。
6. 竖屏短视频8至12支。
7. 精选照片100张以上。
8. 承狮村麒麟舞数字档案目录1套。
9. “歙县北站至承狮村非遗体验”路线方案1套。
10. 面向地方的短视频传播手册和30天发布日历1套。

## 项目特色

第一，选题具体。项目聚焦富堨镇承狮村麒麟舞，不停留在笼统的歙县非遗介绍。

第二，方法复合。项目同时开展文化调研、影像记录、平台分析和路线设计。

第三，北邮特色鲜明。项目突出数字媒体、网络传播、数据分析和二维码导览等信息化手段。

第四，成果可落地。项目不只完成报告，还将向地方交付可发布、可展示、可运营、可继续更新的成果包。
