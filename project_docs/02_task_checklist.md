# 任务清单与执行分工

## 一、5人角色设置

| 角色 | 人数 | 核心任务 |
|---|---:|---|
| 项目负责人兼地方联络 | 1 | 学校申报、地方对接、日程统筹、授权确认、风险决策 |
| 调研与文字负责人 | 1 | 访谈提纲、问卷、田野笔记、调研报告、论文初稿 |
| 影像与后期负责人 | 1 | 拍摄、收音、照片、素材备份、短片剪辑 |
| 传播与数据负责人 | 1 | 短视频样本分析、选题标题、平台发布建议、数据复盘 |
| 安全财务与路线负责人 | 1 | 预算、票据、保险、药品、路线试走、应急通讯 |

说明：5人小队不再拆成多个小组。实地期间所有成员都要协助访谈、搬运设备、问卷和素材整理。

## 三、实地每日任务

## 五、每日复盘模板

1. 今天完成了什么？
2. 哪些信息得到核实？
3. 哪些表述仍不能下结论？
4. 明天需要补拍或补访什么？
5. 素材是否完成双备份？
6. 是否出现安全、授权、文化表达风险？
