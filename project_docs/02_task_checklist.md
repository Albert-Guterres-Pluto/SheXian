# 任务清单与执行分工

## 一、角色设置

| 角色 | 建议负责人 | 核心任务 |
|---|---|---|
| 总负责人 | 1人 | 学校申报、地方对接、日程统筹、风险决策 |
| 地方联络 | 1人 | 对接歙县文旅部门、富堨镇、承狮村、传承人 |
| 调研负责人 | 1人 | 访谈提纲、问卷、田野笔记、调研报告 |
| 影像导演 | 1人 | 宣传片脚本、分镜、拍摄调度 |
| 摄影摄像 | 2人 | 主机位、辅机位、照片、素材备份 |
| 收音与场记 | 1人 | 录音、场记表、授权记录 |
| 数据与传播 | 2人 | 短视频样本分析、账号运营日历、数据复盘 |
| 路线设计 | 1至2人 | 路线试走、点位记录、商户访谈 |
| 财务安全 | 1人 | 预算、票据、保险、药品、应急通讯 |

## 三、实地每日任务

### D1 抵达与踏勘

| 任务 | 负责人 | 交付物 |
|---|---|---|
| 拜访富堨镇、承狮村 | 总负责人、地方联络 | 会议纪要 |
| 核实麒麟舞传承人、队员和活动场地 | 调研组 | 对象清单 |
| 歙县北站至承狮村路线试走 | 路线组 | 时间、距离、交通方式、拍摄点 |
| 拍摄环境空镜 | 影像组 | 村口、道路、站点、标识素材 |

### D2 传承人深访

| 任务 | 负责人 | 交付物 |
|---|---|---|
| 传承人长访 | 调研负责人 | 录音、文字纪要 |
| 道具服饰拍摄 | 影像组 | 道具照片、细节视频 |
| 麒麟舞动作记录 | 调研组、影像组 | 动作名称、动作含义、视频编号 |
| 肖像和素材授权 | 场记 | 授权记录 |

### D3 村落空间与村民访谈

| 任务 | 负责人 | 交付物 |
|---|---|---|
| 记录祠堂、广场、巷道等表演空间 | 路线组、影像组 | 点位图、照片 |
| 访谈老年村民 | 调研组 | 访谈记录 |
| 访谈青少年和返乡青年 | 调研组 | 青年参与分析 |
| 拍摄“麒麟舞在哪里发生”短片素材 | 影像组 | 空间叙事素材 |

### D4 传承困境与地方需求

| 任务 | 负责人 | 交付物 |
|---|---|---|

### D5 县域联动调研

| 任务 | 负责人 | 交付物 |

## 五、每日复盘模板

1. 今天完成了什么？
2. 哪些信息得到核实？
3. 哪些表述仍不能下结论？
4. 明天需要补拍或补访什么？
5. 素材是否完成双备份？
6. 是否出现安全、授权、文化表达风险？
