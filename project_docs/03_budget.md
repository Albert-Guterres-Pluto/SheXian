# 经费预算建议

预算按5名队员、5至6天实地实践估算。金额可根据学校报销标准、实际交通票价和地方接待情况调整。

| 当地交通 | 高铁站至村镇接驳、县域调研、必要包车 | 1800元 |

## 三、不建议列入本次预算的内容

本项目为5人轻量实践，不建议在立项阶段承诺或预算以下内容：

1. 大型成果展。
2. 大批量文创产品。
3. 专业拍摄团队外包。
4. 长周期账号代运营。
5. 双语宣传片和复杂互动网页。

## 四、费用使用原则

1. 与传承人、表演队相关的劳务支持要提前说明标准，避免临时议价。
2. 能开票的住宿、交通、打印、物料优先开具正规票据。
3. 展演支持、访谈补贴如无法开票，应保留签收记录、事由说明和照片佐证。
4. 每日由安全财务负责人登记支出，项目负责人复核。
5. 不采购与项目无关的纪念品，不把经费用于高消费餐饮。
6. 机动费用只用于安全、交通、设备、天气调整等必要事项。
