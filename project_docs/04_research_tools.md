# 调研工具包

## 一、访谈通用要求

1. 访谈前说明身份、项目目的、使用范围和是否公开发布。
2. 录音、录像、拍照前必须获得明确同意。
3. 不诱导受访者给出“项目想听的答案”。
4. 对历史源流、传说、名录级别等内容，采用“据受访者回忆”“待进一步核实”等表述。
5. 每次访谈结束后记录时间、地点、受访者身份、核心观点和待核实问题。

## 二、传承人访谈提纲

基本信息：

1. 您的姓名、年龄、在麒麟舞队中的角色是什么？
2. 您从什么时候开始接触麒麟舞？是谁教您的？
3. 当前承狮村还有多少人会跳麒麟舞？年龄结构如何？

历史与源流：

1. 村里关于麒麟舞的最早记忆是什么？
2. 麒麟舞与承狮村村名、村落历史、节庆习俗是否有关？
3. 过去哪些时候会跳麒麟舞？春节、庙会、婚庆、庆典还是其他场合？
4. 有没有村志、族谱、老照片、旧道具或文字资料可以印证？

表演内容：

1. 麒麟舞由几个人完成？有哪些角色？
2. 主要动作有哪些？每个动作有什么寓意？
3. 锣鼓节奏是否有固定曲牌或口传口诀？
4. 麒麟头、服饰、道具由谁制作和维护？
5. 表演时有哪些禁忌或规矩？

传承现状：

1. 现在多久排练一次？经费从哪里来？
2. 年轻人愿意学吗？最大的困难是什么？
3. 表演机会主要来自哪里？
4. 您希望政府、学校、媒体或社会团队提供什么帮助？

传播与转化：

1. 您是否愿意通过短视频传播麒麟舞？
2. 哪些内容可以公开拍摄？哪些内容不适合公开？
3. 如果游客来承狮村体验，您认为可以体验哪些环节？
4. 您希望视频突出“热闹好看”还是“历史文化”，或者两者结合？

## 三、村干部和文化站访谈提纲

1. 承狮村麒麟舞目前是否有正式非遗名录身份？若有，属于哪一级？
2. 村里是否保存相关申报材料、照片、视频或文字资料？
3. 当前传承队的组织方式是什么？由谁管理？
4. 政府或村集体是否曾经组织展演、培训、进校园活动？
5. 麒麟舞与富堨镇、歙县文旅发展之间目前有哪些连接？
6. 如果设计“歙县北站至承狮村非遗体验路线”，现实阻碍是什么？
7. 当地最需要团队交付哪类成果：宣传片、素材库、调研报告、路线方案、申报材料补充，还是账号运营建议？

## 四、村民访谈提纲

1. 您小时候见过麒麟舞吗？通常在什么场合？
2. 您觉得麒麟舞对承狮村意味着什么？
3. 现在村里年轻人还了解麒麟舞吗？
4. 如果游客来村里看麒麟舞，您支持吗？担心什么？
5. 您认为承狮村还有哪些地方适合游客了解村庄？

## 五、游客和青年受众问卷框架

题目建议控制在3分钟内完成。

1. 您的年龄：18岁以下 / 18-25岁 / 26-35岁 / 36-50岁 / 50岁以上
2. 您是否来过歙县：来过 / 第一次来 / 正计划前往 / 只是线上了解
3. 您知道歙县哪些文化资源：徽州古城 / 渔梁坝 / 棠樾牌坊群 / 徽墨 / 歙砚 / 新安江山水画廊 / 其他
4. 您是否听说过承狮村麒麟舞：听说过且看过 / 听说过但没看过 / 没听说过
5. 如果从歙县北站出发可到村里看麒麟舞，您是否愿意参加：愿意 / 看时间和价格 / 不太愿意
6. 您能接受的非遗体验时长：15分钟 / 30分钟 / 1小时 / 半天
7. 您最想看到什么：完整表演 / 传承人讲故事 / 学一个动作 / 道具制作 / 村落导览 / 拍照打卡
8. 您通常在哪个平台看文旅或非遗内容：抖音 / 视频号 / 小红书 / B站 / 微博 / 其他
9. 什么样的标题会吸引您点击：视觉奇观 / 人物故事 / 路线攻略 / 历史知识 / 互动挑战
10. 您更能接受哪种非遗体验组织方式：公益展示 / 预约参观 / 研学课程配套 / 节庆活动配套 / 暂不参加
11. 您是否愿意转发承狮村麒麟舞短视频：愿意 / 视内容而定 / 不愿意
12. 您对承狮村麒麟舞传播有什么建议？开放题。

## 六、田野观察记录表

| 项目 | 记录内容 |
|---|---|
| 时间 | 年月日、开始结束时间 |
| 地点 | 村口、广场、祠堂、巷道、室内排练点等 |
| 参与者 | 传承人、队员、村民、游客、干部 |
| 事件 | 排练、展演、访谈、道具整理、路线试走 |
| 动作 | 动作名称、队形变化、节奏变化、寓意 |
| 声音 | 锣鼓节奏、口令、现场交流 |
| 空间 | 表演空间大小、观众位置、出入口、拍摄机位 |
| 互动 | 村民反应、游客反应、儿童参与 |
| 问题 | 现场困难、表达不清、需补访事项 |
| 素材编号 | 对应照片、视频、录音文件名 |

## 七、素材授权要点

授权文件至少包含：

1. 受访者姓名和联系方式。
2. 授权内容：录音、录像、照片、文字整理。
3. 使用范围：学校社会实践、调研报告、论文、短视频、宣传片、地方文旅宣传。
4. 是否允许公开姓名和肖像。
5. 是否允许剪辑后在互联网平台发布。
6. 未成年人须由监护人同意。
7. 授权日期和双方签字。

## 八、样本量和结项材料要求

1. 5人小队建议完成10至15份有效访谈，其中至少包括传承人或表演队代表、村干部或文化站工作人员、普通村民、游客或青年受众。
2. 问卷目标为80至120份，优先保证样本结构清楚、数据真实，不为追求数量重复填报。
3. 调研报告需真实、客观、可信，包含标题、摘要、调研背景、问题分析、对策建议和参考文献。
4. 访谈、问卷和照片视频都要能对应到具体时间、地点和任务，便于结项登记表和成果审核追溯。
