# 实践路线与地方价值设计

## 一、路线设计原则

1. 以承狮村麒麟舞为核心，不把它变成歙县大景区的附属背景。
2. 利用歙县北站区位，将承狮村设计为“抵达歙县后的非遗第一站”。
3. 路线要考虑游客真实需求：交通、停车、卫生间、餐饮、讲解、拍照点、停留时间。
4. 对村落承载力保持克制，优先设计小规模预约式体验。
5. 路线不只服务游客，也服务研学、学校活动、地方节庆和官方推介。

## 二、半日线：歙县北站至承狮村麒麟舞体验

目标人群：高铁抵达游客、亲子家庭、研学小队、短暂停留游客。

建议时长：2至3小时。

路线结构：

1. 歙县北站集合。
2. 前往承狮村村口，进行村落导入。
3. 参观麒麟舞道具和服饰。
4. 传承人讲述麒麟舞源流和寓意。
5. 观看15至20分钟展演或排练展示。
6. 学习一个简单动作或锣鼓节奏。
7. 在村口或公共空间拍摄“我在承狮遇见麒麟”短视频。
8. 领取电子明信片或扫码查看导览页。

需要核实：

1. 歙县北站至承狮村实际交通时间。
2. 村内可停车位置。
3. 是否有适合小型展演和游客停留的公共空间。
4. 展演频率和传承人可参与时间。

## 三、一日线：徽州古城+承狮村麒麟舞

目标人群：第一次到歙县的游客。

建议时长：6至8小时。

路线结构：

1. 上午：徽州古城或渔梁坝，建立游客对徽州历史文化的基本认知。
2. 中午：县城或富堨镇用餐。
3. 下午：承狮村麒麟舞体验。
4. 傍晚：返回歙县北站或县城住宿点。

传播主题：

1. “歙县不只有古城，还有会起舞的村落记忆。”
2. “上午看徽州古建，下午看承狮麒麟。”
3. “从城到村，理解一个更完整的徽州。”

## 四、两日研学线：徽州技艺+承狮麒麟舞

目标人群：高校、中小学研学团队、文化社团。

建议结构：

第一天：

1. 徽州古城或渔梁坝导览。
2. 徽墨、歙砚或徽派工艺体验。
3. 晚间完成“徽州文化关键词”小组讨论。

第二天：

1. 歙县北站或县城出发至承狮村。
2. 传承人讲述麒麟舞。
3. 学生分组完成动作学习、道具观察、人物访谈、短视频拍摄。
4. 小组展示15秒至30秒短视频。
5. 由传承人和老师点评。

研学任务单：

1. 记录一个动作名称。
2. 采访一位传承人或村民。
3. 画出一张承狮村麒麟舞空间图。
4. 拍摄一条30秒非遗短视频。
5. 写下“我理解的非遗传承不是……”一句话。

## 五、地方交付成果

| 成果 | 交付对象 | 用途 |
|---|---|---|
| 承狮村麒麟舞综合短片 | 承狮村、富堨镇、歙县文旅部门 | 官方推介、活动展示 |
| 3至5条竖屏短视频 | 村委会、镇文化站、地方账号 | 后续转发或二次剪辑 |
| 简版发布建议 | 地方运营人员 | 标题、封面、发布时间参考 |
| 路线建议 | 镇村、旅行社、研学机构 | 判断非遗体验产品可行性 |
| 基础资料目录 | 传承人、文化站 | 保护和申报资料积累 |
| 二维码导览页文案 | 承狮村 | 游客自助了解 |

## 六、二维码导览页建议内容

页面标题：承狮村麒麟舞

模块：

1. 30秒视频：麒麟舞视觉入口。
2. 一句话介绍：承狮村麒麟舞是富堨镇承狮村延续至今的民俗表演传统，具体源流以地方文献和传承人口述为准。
3. 看点一：麒麟头和服饰。
4. 看点二：锣鼓节奏。
5. 看点三：动作寓意。
6. 看点四：村落空间。
7. 到达方式：从歙县北站出发的交通提示。
8. 预约方式：村委会或镇文化站联系方式，待地方确认后填写。
9. 参与提示：尊重村民生活，不擅自拍摄未成年人和宗祠内部。

## 七、实际价值路径

短期价值：

1. 为承狮村留下系统影像资料。
2. 帮助地方形成可发布的短视频内容。
3. 为麒麟舞形成真实、合规的展示素材和传播入口。
4. 为镇村后续申报、展示和活动策划积累基础文字材料。

中期价值：

1. 形成“歙县北站至承狮村”的非遗体验路线。
2. 吸引研学团队、小型旅行团和青年游客。
3. 为本地讲解、展演、餐饮、农产品等文旅配套需求提供参考。

长期价值：

1. 让年轻人看到麒麟舞的公共价值和传播价值，提高参与意愿。
2. 为歙县乡村非遗数字化保护提供可复制样本。
3. 形成高校长期结对实践基地的可能。
