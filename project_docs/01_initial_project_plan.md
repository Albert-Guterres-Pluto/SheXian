# 初步策划方案

1. 承狮有麟：歙县富堨镇麒麟舞文化基因挖掘与传播实践
2. 下高铁看非遗：承狮村麒麟舞短视频出圈计划
3. 麟舞承狮：北邮青年助力徽州非遗数字传播

10. 集邮广益·麟耀承狮：徽州乡村非遗传播行动

## 二、项目背景

歙县是徽文化核心区域，古城、古村落、古牌坊、古祠堂、徽墨、歙砚等文化资源集中。游客对歙县的认知多集中在徽州古城、渔梁坝、棠樾牌坊群、新安江山水画廊等成熟目的地，乡村民俗类资源仍存在“在地有传统、线上缺表达、游客难抵达、体验不成线”的问题。

富堨镇承狮村具有麒麟舞传统。麒麟舞兼具表演性、仪式性、村落记忆和祥瑞寓意，适合进行文化内涵挖掘、影像记录和短视频转化。承狮村还具备区位传播优势：歙县北站位于富堨镇承狮村一带，项目可以将“高铁门户”和“乡村非遗”结合起来，形成“抵达歙县第一眼非遗”的传播叙事。

短视频平台改变了非遗传播逻辑。非遗内容不能只依靠长篇介绍，而要具备视觉钩子、人物故事、可参与动作、地点标签、路线转化和持续更新机制。北京邮电大学在信息通信、数字媒体、网络传播、数据分析等方面具有学科特色，适合以“数字化记录+传播策略+路线共创+地方交付”的方式参与乡村文化振兴。

## 三、实践目标

1. 完成承狮村麒麟舞的田野调查，梳理其历史源流、传承谱系、表演场景、道具服饰、锣鼓音乐、动作结构和村落关系。
2. 建立承狮村麒麟舞数字档案目录，形成照片、视频、访谈文字稿、动作记录、路线点位等可持续使用资料。
3. 拍摄1支3至5分钟宣传片、8至12支竖屏短视频，形成适合抖音、视频号、小红书、B站和学校官方平台的内容矩阵。
4. 设计“歙县北站至承狮村麒麟舞体验”特色路线，并与徽州古城、渔梁坝、徽墨歙砚体验等县域资源形成联动。
5. 形成论文、调研报告、采访记录、照片视频、短视频传播手册、路线方案等成果，为当地后续非遗保护、研学活动和文旅传播提供实际材料。

## 四、研究问题

| 编号 | 研究问题 | 需采集证据 |
|---|---|---|
| RQ1 | 承狮村麒麟舞从何而来，经历过哪些传承变化？ | 传承人口述、村志族谱线索、文化站资料、老年村民访谈 |
| RQ2 | 麒麟舞与承狮村村落空间、节庆习俗、集体记忆之间有何关系？ | 表演地点、节庆节点、祠堂/广场/巷道观察记录 |
| RQ3 | 当前承狮村麒麟舞面临哪些传承难题？ | 传承队年龄结构、训练频率、经费来源、年轻人参与度 |
| RQ4 | 短视频平台上非遗内容如何获得传播？ | 同类账号样本、标题封面、时长、互动数据、用户评论 |
| RQ5 | 承狮村麒麟舞能否转化为歙县乡村文旅和研学资源？ | 路线试走、游客问卷、商户访谈、镇村反馈 |
| RQ6 | 北邮团队能提供哪些可持续的数字化支持？ | 素材库、二维码导览、账号运营日历、数据复盘模型 |

## 五、实践路线

建议实地时间为7天，人数按10至12人配置。

| 时间 | 地点 | 主要任务 | 当日产出 |
|---|---|---|---|
| D1 | 歙县北站、承狮村、富堨镇 | 抵达踏勘，拜访镇村干部，确认传承人和拍摄安排 | 联系人表、点位图、授权清单 |
| D2 | 承狮村 | 传承人深访，记录源流、谱系、动作、道具、锣鼓 | 访谈录音、人物照片、初步口述史 |
| D3 | 承狮村村落空间 | 观察表演空间、祠堂广场巷道、村民日常 | 村落影像、空间叙事笔记 |
| D4 | 富堨镇及周边 | 访问文化站、学校、商户、返乡青年 | 问卷数据、传承困境清单 |

| D6 | 承狮村 | 集中拍摄宣传片、竖屏短视频、互动体验内容 | 主片素材、短视频素材 |

## 六、调研对象

1. 政府和机构：歙县文旅体局、歙县文化馆、富堨镇政府、富堨镇文化站。
2. 传承主体：承狮村麒麟舞传承人、表演队负责人、队员、锣鼓师、道具服饰维护者。
3. 村落主体：承狮村村委会、老年村民、普通村民、返乡青年、青少年。
4. 文旅主体：歙县北站周边经营者、民宿餐饮经营者、导游、旅行社或研学机构。
5. 传播主体：歙县融媒体、本地文旅账号运营者、短视频创作者。
6. 受众主体：歙县游客、本地学生、北邮学生及线上青年受众。

## 七、调研方法

1. 口述史访谈：围绕“谁传下来的、何时表演、在哪里表演、如何训练、现在难在哪里”展开。
2. 参与式观察：跟拍排练或展演，记录动作队形、音乐节奏、角色分工和村民互动。
3. 视觉民族志：用照片、视频和空间记录呈现麒麟舞与承狮村空间的关系。
4. 问卷调查：面向村民、游客和青年受众，测量认知度、观看兴趣、参与意愿和路线接受度。
5. 短视频内容分析：收集同类非遗短视频样本，分析标题、封面、首屏画面、时长、评论和互动机制。
6. 路线试走：记录从歙县北站到承狮村主要点位的交通时间、停车点、公共服务设施、网络信号和最佳拍摄点。
7. 共创工作坊：邀请村干部、传承人、青年和商户共同观看初剪视频，修正不准确表达。

## 八、创新点

1. 以承狮村麒麟舞为具体个案，避免泛泛谈非遗，形成可验证、可交付、可持续更新的村落样本。
2. 将歙县北站区位转化为传播入口，提出“高铁门户非遗体验”路线。
3. 采用“长片讲深度、短片做传播、素材库留地方、路线促转化”的组合成果。
4. 利用北邮的信息通信和数字媒体特色，开展网络条件观察、短视频数据分析、二维码导览和平台运营设计。
5. 将非遗保护与乡村振兴结合，把麒麟舞从节庆展演拓展为研学体验、游客路线和地方品牌内容。
