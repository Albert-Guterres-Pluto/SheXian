# 初步策划方案

## 二、项目背景

歙县是徽文化核心区域，古城、古村落、古牌坊、古祠堂、徽墨、歙砚等文化资源集中。游客对歙县的认知多集中在徽州古城、渔梁坝、棠樾牌坊群、新安江山水画廊等成熟目的地，乡村民俗类资源仍存在“在地有传统、线上缺表达、游客难抵达、体验不成线”的问题。

富堨镇承狮村具有麒麟舞传统。麒麟舞兼具表演性、仪式性、村落记忆和祥瑞寓意，适合进行文化内涵挖掘、影像记录和短视频转化。承狮村还具备区位传播优势：歙县北站位于富堨镇承狮村一带，项目可以将“高铁门户”和“乡村非遗”结合起来，形成“抵达歙县第一眼非遗”的传播叙事。

本项目暂定为5人小队，成果设置坚持“少而实”：不追求大规模展陈和复杂成果包，重点完成文化调研、基础影像记录、少量短视频传播和一份可供当地参考的路线建议。北京邮电大学团队可发挥信息通信、数字媒体、网络传播和数据分析优势，以轻量化方式参与乡村文化振兴。

## 三、实践目标

1. 完成承狮村麒麟舞基础田野调查，梳理其历史源流、传承谱系、表演场景、道具服饰、锣鼓音乐、动作结构和村落关系。
2. 建立轻量化数字资料目录，整理照片、视频、访谈文字稿、动作记录和路线点位，便于后续继续补充。
3. 拍摄1支2至3分钟横屏综合宣传短片、3至5支竖屏短视频，形成适合学校结项、校内展示和地方账号转发的基础内容。
4. 设计“歙县北站至承狮村麒麟舞体验”简版特色路线，并提出与徽州古城、渔梁坝等县域资源联动的建议。
5. 完成论文或论文式实践总结、调研报告、采访记录、照片视频等核心成果，为当地后续非遗保护和文旅传播提供实用材料。

## 四、研究问题

| 编号 | 研究问题 | 需采集证据 |
|---|---|---|
| RQ1 | 承狮村麒麟舞从何而来，经历过哪些传承变化？ | 传承人口述、村志族谱线索、文化站资料、老年村民访谈 |
| RQ2 | 麒麟舞与承狮村村落空间、节庆习俗、集体记忆之间有何关系？ | 表演地点、节庆节点、祠堂/广场/巷道观察记录 |
| RQ3 | 当前承狮村麒麟舞面临哪些传承难题？ | 传承队年龄结构、训练频率、经费来源、年轻人参与度 |
| RQ4 | 短视频平台上非遗内容如何获得传播？ | 同类账号样本、标题封面、时长、互动数据、用户评论 |
| RQ5 | 承狮村麒麟舞能否转化为歙县乡村文旅和研学资源？ | 路线试走、游客问卷、商户或干部访谈、镇村反馈 |

## 五、实践路线

建议实地时间为5至6天，团队人数暂定5人。若地方展演时间难以协调，可将D4、D5合并，压缩为5天。正式出发前应完成学校审批、安全培训、保险购买、成员安全责任书和家长/监护人知情同意书收集。

| 时间 | 地点 | 主要任务 | 当日产出 |
|---|---|---|---|
| D1 | 歙县北站、承狮村、富堨镇 | 抵达踏勘，拜访镇村干部，确认传承人和拍摄安排 | 联系人表、点位图、授权清单 |
| D2 | 承狮村 | 传承人深访，记录源流、谱系、动作、道具、锣鼓 | 访谈录音、人物照片、口述史要点 |
| D3 | 承狮村村落空间 | 观察表演空间、祠堂广场巷道、村民日常 | 村落影像、空间叙事笔记 |
| D4 | 富堨镇及周边 | 访问文化站、学校或商户，发放简版问卷 | 问卷数据、传承与传播问题清单 |
| D5 | 歙县县城或游客点位 | 调研游客对“从高铁站到承狮村看非遗”的接受度 | 游客反馈、路线转化分析 |
| D6 | 承狮村或富堨镇 | 补拍关键画面，展示初剪片段并收集修改意见 | 视频初剪、反馈表、后续联络机制 |

## 六、调研对象

1. 政府和机构：歙县文旅体局、歙县文化馆、富堨镇政府、富堨镇文化站。
2. 传承主体：承狮村麒麟舞传承人、表演队负责人、队员、锣鼓师、道具服饰维护者。
3. 村落主体：承狮村村委会、老年村民、普通村民、返乡青年、青少年。
4. 文旅主体：歙县北站周边经营者、民宿餐饮经营者、导游或研学机构。
5. 受众主体：歙县游客、本地学生、北邮学生及线上青年受众。

## 七、调研方法

1. 口述史访谈：围绕“谁传下来的、何时表演、在哪里表演、如何训练、现在难在哪里”展开。
2. 参与式观察：跟拍排练或展演，记录动作队形、音乐节奏、角色分工和村民互动。
3. 影像记录：用照片、视频和空间记录呈现麒麟舞与承狮村空间的关系。
4. 简版问卷调查：面向村民、游客和青年受众，测量认知度、观看兴趣、参与意愿和路线接受度。
5. 短视频内容分析：收集同类非遗短视频样本，分析标题、封面、首屏画面、时长、评论和互动机制。
6. 路线试走：记录从歙县北站到承狮村主要点位的交通时间、停车点、公共服务设施、网络信号和最佳拍摄点。

## 八、创新点

1. 以承狮村麒麟舞为具体个案，避免泛泛谈非遗，形成可验证、可交付的村落样本。
2. 将歙县北站区位转化为传播入口，提出“高铁门户非遗体验”路线。
3. 采用“短片记录、短视频传播、简版路线建议、基础资料留存”的轻量成果结构。
4. 利用北邮的信息通信和数字媒体特色，开展网络条件观察、短视频数据分析和二维码导览文案设计。
5. 将非遗保护与乡村振兴结合，把麒麟舞从节庆展演拓展为游客路线和地方品牌内容。

## 十、团队分工

5人小队采用“一人主责、多岗协同”的方式，不设置过多小组。

| 岗位 | 人数 | 主要职责 |
|---|---:|---|
| 项目负责人兼地方联络 | 1 | 对接学校、镇村、传承人，统筹进度、授权和质量 |
| 调研与文字负责人 | 1 | 访谈、问卷、田野笔记、调研报告和论文初稿 |
| 影像与后期负责人 | 1 | 拍摄、收音、素材管理、短片剪辑 |
| 传播与数据负责人 | 1 | 短视频选题、标题封面、平台样本分析、发布复盘 |
| 安全财务与路线负责人 | 1 | 经费台账、保险药品、路线试走、每日考勤和应急联络 |
